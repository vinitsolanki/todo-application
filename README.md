# todo-application

Build backend app (todo-app)

1. `cd todo-app`
2. `./gradlew bootRun` to Run `TodoAppApplication.java` as java application 


Build Frontend app (todo-angular-app)
1. `npm install`
2. `npm start`

frontend application will start on http://localhost:9090
